# Holiday Traditions Bot - Обзор проекта

## Что это?

Telegram-бот, который генерирует интересные факты о праздновании Рождества и Нового года в 50+ странах мира с помощью AI (OpenAI GPT).

## Созданные файлы

### Основные файлы (Python)
- **bot.py** (8.0 KB) — основная логика Telegram бота
- **llm.py** (2.5 KB) — интеграция с OpenAI API
- **config.py** (2.1 KB) — конфигурация и системный промпт
- **countries.py** (1.5 KB) — список из 50+ стран

### Конфигурация
- **requirements.txt** — зависимости Python
- **.env.example** — шаблон для переменных окружения
- **.gitignore** — правила для Git

### Документация
- **README.md** (7.5 KB) — полная документация
- **QUICKSTART.md** (2.3 KB) — быстрый старт
- **ARCHITECTURE.md** (7.8 KB) — архитектура проекта
- **PROJECT_SUMMARY.md** — этот файл

## Что делает бот?

По команде пользователя генерирует карточку праздника, включающую:

- Название страны и праздника
- Традиционные наряды
- Обычаи и блюда
- Рекомендацию фильма
- Рекомендацию музыки
- Совет по адаптации традиции дома

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и справка |
| `/holiday` | Случайная страна и праздник |
| `/christmas` | Рождество в случайной стране |
| `/newyear` | Новый год в случайной стране |
| `/another` | Повторить с другой страной |

## Технологии

- **Python 3.8+**
- **python-telegram-bot** — Telegram Bot API
- **OpenAI API** — GPT-4o-mini для генерации
- **python-dotenv** — управление секретами

## Быстрый запуск

```bash
# 1. Установка
cd holiday_bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Настройка
cp .env.example .env
# Отредактируйте .env и добавьте токены

# 3. Запуск
python bot.py
```

## Что нужно для работы?

1. **Telegram Bot Token** — получить у [@BotFather](https://t.me/botfather)
2. **OpenAI API Key** — получить на [platform.openai.com](https://platform.openai.com)
3. Python 3.8 или выше
4. Доступ к интернету

## Безопасность

- Токены хранятся в `.env` (не в коде)
- `.env` добавлен в `.gitignore`
- Нет прямого пользовательского ввода в LLM
- Лимиты на длину генерации (max_tokens=1000)
